PSQT Editor

This is a rudimentary editor made in Xtext. This file is easier to maintain since it is not code.

Generate code

By rightclicking the file in the navigator you will have options to generate a code file. The code file has the Stockfish C++ syntax, but that works well also with Java, like my own engine.

Generate graphics

By rightclicking the file in the navigator you will have options to generate a graphics for the file. This graphics will show boards with colours that indicate bad or good squares.
